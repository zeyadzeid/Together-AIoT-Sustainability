# Together: AIoT and Mobile Solution for Climate Resilience

Together is an integrated open-source system designed to combat climate change impacts on agriculture and enhance community awareness. This project was featured at COP28 UAE as a youth-led innovation for sustainability.

## Key Components
1. Empowering Mobile App: Features a CO2 tracker, environmental habit building, and a community platform.
2. AIoT Smart Farming: A closed-loop IoT system with a Smart Planting Guide to optimize water usage and crop health.

## Repository Structure
* /mobile-app-demo: Contains the latest functional APK for testing.
* /docs: Technical reports, business models, and presentations (COP28).
* /iot-logic: (Coming Soon) Embedded C++ code for sensor integration.

## Tech Stack
* Software: Flutter/Dart (Mobile), Python (Data Analysis).
* Hardware: ESP32, Arduino, Soil & Humidity Sensors.
* Impact: Aligned with UN SDGs 2, 12, and 13.

## Open Source Status
We are currently in the process of refactoring our internal code for full public contribution. Technical documentation and the demo APK are provided for immediate review and collaboration.
